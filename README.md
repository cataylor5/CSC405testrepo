## Project Title
Analysis of Dirty Iranian Transactions Dataset

## Dataset
Dataset source:  
(https://www.kaggle.com/datasets/hosseinbadrnezhad/dirty-iranian-transactions-dataset?select=trx-10k.csv)

## Project Description
This project applies the full data science pipeline to a real-world dataset.
We will perform data cleaning, exploratory data analysis, statistical testing,
modeling, and visualization to investigate a data-driven problem defined by the team.

## Repository Structure
data/        Dataset files or dataset instructions  
notebooks/   Shared Jupyter notebooks for analysis  
src/         Helper code and functions  
figures/     Generated plots and figures  
report/      Final report notebook or exported report  

## How to Run
1. Clone the repository:
   git clone https://github.com/cataylor5/CSC405-Group-Project.git
2. Open the project in VS Code or Jupyter.
3. Run notebooks in the notebooks/ folder in order.
